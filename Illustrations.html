Illustrations

This project will be using the skills I learned in Digital Illustration to create a skull shaped halloween mask.

These include: creating a new document, working with artboards, selecting and aligning objects, using basic shapes, using the shape builder tool, copying & duplicating items, shaper & shape builder tools, rotate and reflecting objects, pathfinder panel & combining shapes, distort & transform effects locking objects, drawing modes, and eyedropper tool.
